Affichez les annonces de voiture en provenance de la plateforme Annonces Automobile

== Description ==

Annonces Automobile est une plateforme sur laquelle vous trouverez plusieurs milliers d'annonces de voiture d'occasion ou neuve. Grâce au plugin Wordpress Annonces Automobile, vous pouvez très facilement afficher les annonces proposées par la plateforme sur votre site Internet. L'affichage peut être effectué par catégorie (Break, berline, 4x4, cabriolet, etc...) ou par marque de voiture. Si vous êtes déjà client professionnel chez Annonces Automobile, vous pouvez aussi rapidement afficher votre stock de véhicules en quelques clics.

De plus, grâce au plugin Wordpress, vous pouvez être rémunéré en mettant en place le formulaire de dépôt d'annonce sur votre site Internet. Pour chaque paiement réalisé par le biais du formulaire, vous pouvez être rémunéré à hauteur de 50% ! Notre système de monétisation est l'un des plus rentable parmi tous les autres systèmes d'affiliation ou de rémunération.

Attention, le plugin Annonces Automobile s'appuie sur un tiers et effectue des appels de flux XML en provenance de https://www.annonces-automobile.com en tant que service. Lorsque vous utilisez le module Annonces Automobile, les données fournies (Annonces de voiture, photos et coordonnées des acheteurs) proviennent de la plateforme Annonces-Automobile.com qui en est le propriétaire exclusif. En outre, en utilisant le plugin des données peuvent être transmises vers la plateforme Annonces Automobile et conservées sur son serveur.

Pour plus d'informations, vous pouvez consulter la pages dédiée aux conditions d'utilisation et aux informations légales d'Annonces Automobile directement depuis ce lien : https://www.annonces-automobile.com/mentions-legales

== Installation ==

1. Téléchargez le plugin depuis votre administration Wordpress ou transférez le dans un nouveau dossier dans le répertoire `/wp-content/plugins/`
2. Activez le plugin depuis votre administration
3. Configurez le plugin depuis votre administration

== Frequently asked questions ==

= Le module Annonces Automobile est-il gratuit ? =

Oui, le module est gratuit et accessible pour tous.

= Comment télécharger le module Annonces Automobile ? =

Pour télécharger le module Annonces Automobile vous pouvez vous rendre sur la rubrique ajouter une extension dans votre administration Wordpress
